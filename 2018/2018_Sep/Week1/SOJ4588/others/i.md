Time Limit: 3000 MS　　Memory Limit: 256 M 

***
## Description
　　Sidney想去Gandtom家玩。但Sidney家和Gandtom家之间是高低不平、坑坑洼洼的土路。所以他需要用他的背包装几袋稀的泥，在路上铺平一些干的土，使路变成平整的泥土，才能到Gandtom家见到Gandtom。
　　已知这里有n个结点与m条无向路。Sidney住在1号结点，Gandtom住在n号结点。每条路径会连接某两个结点，并且每条路径都有一个高度$(0 \le h_i \le 2)$。 你能对土路进行以下某种操作：
　　1.对于高度为0的干的土，你可以使用一个单位的稀的泥，使它的高度变为1。
　　2.对于高度为0的干的土，你可以使用两个单位的稀的泥，使它的高度变为2。
　　3.对于高度为1的干的土，你可以使用一个单位的稀的泥，使它的高度变为2。
　　Sidney需要经过一段连续的高度为1的路径（可以为空），然后再经过一段连续的高度为2的路径（可以为空）到达Gandtom家。换句话说，Sidney不能走高度为0的路径，至多只能上一次坡，不能下坡。
　　试求Sidney最少要使用多少个单位的稀的泥才能到Gandtom家。题目保证答案存在。

## Input
第一行有一个整数$T$，表示组数。
每组数据第一行有两个正整数n, m$(0 < n, m \le 10^6)$。
接下来有m行，每行有三个整数$u_i, v_i, h_i$(1 \le u_i, v_i \le n, 0 \le h_i \le 2)$。表示结点$u_i$与结点$v_i$之间有一条高度为$h_i$的无向路。

## Output
每组数据输出一个整数，表示Sidney最少要使用多少个单位的稀得泥才能到Gandtom家。

## Sample Input
1
5 5
1 2 1
2 3 0
3 5 2
1 4 2
4 5 1

## Sample Output
1

## Note
将第2条路径的高度从0变为1，或者将第5条路径的高度从1变为2。
