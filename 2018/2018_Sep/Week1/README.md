**HIHO1814 离散化+并查集**
在2D平面的整点上有N个球，其中第i个球的坐标是(Xi, Yi)。
只要任意平行于坐标轴的直线上有2个球，小Hi就可以取走其中一个。
请你计算小Hi最多可以取走几个球？
solution:
答案等于 N-连通块的个数
离散化后对于每个横坐标,纵坐标开一个vector,把相应的点传进去
后用并查集处理连通块的问题.
